# Google Business Profile — setup checklist

Use this when creating or claiming your profile. Tick each item when done.

---

## Profile basics

- [ ] Profile created or claimed at [business.google.com](https://business.google.com)
- [ ] **Business name** set (e.g. Boca Raton TV Mounting or Elite TV Mounting Boca Raton)
- [ ] **Primary category:** TV Mounting Service
- [ ] **Secondary categories:** Home Theater Installation, Handyman, Audio Visual Equipment Installation (as applicable)
- [ ] **Description** filled (Boca Raton, same-day, concealed wiring, soundbar)
- [ ] **Service area** added: Boca Raton + Delray Beach, Parkland, Weston, Coral Springs, Fort Lauderdale, Palm Beach Gardens, Jupiter
- [ ] **Hours** set (e.g. 8 AM – 8 PM)
- [ ] **Phone** and **website** (landing page URL) added and correct

## Content & attributes

- [ ] **Attributes** set: Same-day service, Online estimates (if true), etc.
- [ ] **Products/Services** added: TV mounting, concealed wiring, soundbar installation, etc., with short descriptions
- [ ] **At least 10–15 photos** uploaded (then keep adding from each job)
- [ ] **Q&A** started with common questions (use `qa-suggested.md`)

## Ongoing

- [ ] **Review link** saved; use `review-script.md` after every job
- [ ] **Weekly post** cadence started (use `weekly-posts-templates.md`)
- [ ] NAP (name, address, phone) matches website and any other listings
